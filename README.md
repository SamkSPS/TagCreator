* **Instant Download:** Exports directly as a `.png` file with transparency, ready to be dropped into your pack.

##  Technologies

* **HTML5 Canvas:** Graphical engine for real-time rendering.
* **JavaScript (Vanilla):** Pixel drawing logic, color manipulation, and font mapping.
* **CSS3:** Responsive control interface and modern UI.

##  How to Use

1.  Access the project link via GitHub Pages.
2.  In the **Format and Text** field, type the content for your tag.
3.  Configure your gradient colors and choose the propagation direction.
4.  If necessary, adjust the width (PX Largura) to fit your specific needs.
5.  Click **DOWNLOAD** (BAIXAR) and move the file to your Minecraft texture folder.
